# Face Recognition Attendance App

Flutter based mobile application that marks attendance using Faces of known people.
---
## Features
- Register new faces quickly with front camera
- Recognize face and mark attendance
- Show table of marked attendance and history
- Can Download CSV file
- Fast and accurate
